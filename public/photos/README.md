# Billeder til hjemmesiden

Forside-banner: Gem et billede her som **hero.jpg** (JPEG).

Sådan får du hero-billedet fra din HEIC-fil:

1. Åbn `photos/IMG_7898.HEIC` i **Preview** (Mac).
2. Vælg **Fil → Eksporter…**
3. Format: **JPEG**. Gem filen som **hero.jpg** i denne mappe (`public/photos/`).

Så vises tipvogn-billedet stort på forsiden.
